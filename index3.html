Tiled Image

Description

The image above is actually 9 distinct pieces, each its own img element.

The purpose is to demonstrate the advanced scaling/cropping that can be accomplished dynamically.

The left image has only been scaled and cropped. The right image has a sharpening filter applied.

  1. The image was scaled to 10% of its original size.
  2. Then each was cropped to its region.
  3. Lastly a minor sharpen filter was run on each.
